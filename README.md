# CLI tool for Downloading images.

## Prerequisites
```
Golang Runtime
```

## 1. Clone Repository
```
git clone https://github.com/ksyuk/Download-WebSite-Images
```

## 2. Run Code
```
go run main.go
```

## 3. Enter info
```
URL:
folder name:
selector:
attribute:
target ext: 
```

### Exmaple
```
URL:https://flower.com/23
folder name:flower
selector:#folwer img
attribute:src
target ext:jpg
```
